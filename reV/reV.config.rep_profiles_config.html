Get the number of representative profiles to save.

Get the reV supply curve column to use for a weighted average in the representative profile meanoid algorithm.

property aggregate_profiles¶

Flag to calculate the aggregate (weighted meanoid) profile for each supply curve point. This behavior is instead of finding the single profile per region closest to the meanoid.
