reV.offshore.cli_offshore module¶

reV-offshore¶

reV Offshore Command Line Interface

reV-offshore [OPTIONS] COMMAND [ARGS]...

Options

-n, --name <name>¶

Job name. Default is “reV-off”.

-v, --verbose¶

Flag to turn on debug logging. Default is not verbose.

direct¶

Main entry point to run offshore wind aggregation

reV-offshore direct [OPTIONS] COMMAND [ARGS]...

Options

-gf, --gen_fpath <gen_fpath>¶

Required reV wind generation/econ output file.

-of, --offshore_fpath <offshore_fpath>¶

Required reV wind farm meta and ORCA cost data inputs.

-pp, --points <points>¶

reV project points to analyze (slice, list, or file string). Default is slice(0, 100)

-sf, --sam_files <sam_files>¶

Required SAM config files (required) (str, dict, or list).

-ld, --log_dir <log_dir>¶

Directory to save offshore logs.

-v, --verbose¶

Flag to turn on debug logging. Default is not verbose.

slurm¶

slurm (Eagle) submission tool for reV supply curve aggregation.

reV-offshore direct slurm [OPTIONS]

Options

-a, --alloc <alloc>¶

Required SLURM allocation account name.

-l, --feature <feature>¶

Additional flags for SLURM job. Format is “–qos=high” or “–depend=[state:job_id]”. Default is None.

-mem, --memory <memory>¶

SLURM node memory request in GB. Default is None

-wt, --walltime <walltime>¶

SLURM walltime request in hours. Default is 1.0

-mod, --module <module>¶

Module to load

-env, --conda_env <conda_env>¶

Conda env to activate

-sout, --stdout_path <stdout_path>¶

Subprocess standard output path. Default is in out_dir.

from-config¶

Run reV offshore aggregation from a config file.

reV-offshore from-config [OPTIONS]

Options

-c, --config_file <config_file>¶

Required reV offshore configuration json file.

-v, --verbose¶

Flag to turn on debug logging. Default is not verbose.

valid-config-keys¶

Echo the valid Offshore config keys

reV-offshore valid-config-keys [OPTIONS]

reV offshore wind farm aggregation module command line interface (CLI).

This module aggregates offshore data from high res wind resource data to coarse wind farm sites and then calculates the ORCA econ data.

Offshore resource / generation data refers to WTK 2km (fine resolution) Offshore farms refer to ORCA data on 600MW wind farms (coarse resolution)
